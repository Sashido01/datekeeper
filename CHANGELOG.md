# Changelog

Все заметные изменения в этом проекте будут документироваться здесь.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),

## [0.1.0] - 2026-03-06
### Добавлено
- Начальная инфраструктура проекта:
  - Создан репозиторий на GitHub (MIT лицензия)
  - Инициализирован Flutter-проект с поддержкой Windows, Linux, Android, iOS
  - Добавлен `.gitignore`
  - Добавлены `LICENSE`, `README.md`, `CHANGELOG.md`
  - Добавлены `PROJECT_CONTEXT.md`, `SECURITY.md`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`
- Выполнен первый коммит.

## [0.2.0] - 2026-03-08
### Добавлено
- Реализован Data Layer (Неделя 1):
  - Доменные модели `Event`, `Category`, `Reminder` с логикой повторений и политикой 29 февраля.
  - Интерфейсы репозиториев.
  - Drift-схема базы данных (таблицы, индексы, миграции, seeding категорий).
  - Конвертеры для перечислений и `DateTime` (UTC).
  - DAO для доступа к данным.
  - Мапперы и реализации репозиториев.
  - Riverpod-провайдеры для внедрения зависимостей.
  - Полный набор тестов (unit и интеграционные), 8 тестов успешно проходят.
